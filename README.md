# 🌸 Waifu Pulls

**The ultimate waifu card collection tracker.** Live prices, grail alerts, museum-style showcase, and the best pulls across 9 platforms.

🔗 **Live:** [hallochupi-sketch.github.io/waifu-pulls](https://hallochupi-sketch.github.io/waifu-pulls)

---

## What Is This?

A luxury collector's dashboard for anime TCG cards — built for serious collectors who care about both the art *and* the investment. Private access, no ads, no noise.

## Cards & Platforms

- **71+ curated cards** across 9 platforms
- Weiss Schwarz · Yu-Gi-Oh · Pokémon · One Piece · Dragon Ball Super · Riftbound · Dandadan · VeVe · Signed SP
- S-Tier vault (hero grails), A-Tier gallery, B-Tier rising, C-Tier hidden gems

## Features

- 🏛️ **Museum mode** — luxury dark aesthetic, sakura particles, scroll reveals
- 📈 **Live price sparklines** with 30-day trend data
- 🔮 **Card of the Day / Week** with countdown timers
- 💎 **Collection tracker** — portfolio value, localStorage-based
- 🔔 **Price alerts** — set target prices
- ⭐ **XP & collector levels** — Browser → Collector → Connoisseur → Grail Hunter → Whale
- 🔥 **Visit streaks** with milestone celebrations
- 🎲 **Surprise Me** — discover random cards
- 📡 **Activity feed** — what's moving today
- 🌍 **Geo-smart links** — eBay US, eBay DE, TCGPlayer, Cardmarket, Mercari JP
- 👁️ **Personal watchlist**
- 🔗 **Share system** with deep linking & card-level URLs
- 🔒 **Private access gate** — curated, not public

## Tech

Pure static HTML + CSS + JS. Hosted on GitHub Pages. Zero server dependencies. All user data in localStorage. Price data from `prices.json`. Analytics via CountAPI.
